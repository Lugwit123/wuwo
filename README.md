# Wuwo

Rez package management and configuration tool.

## Features

- Rez configuration management
- Package synchronization
- Custom package loader

## Files

- `config.yaml` - Main configuration file
- `config_loader.py` - Configuration loader
- `rez_comanf_reconfig.py` - Rez configuration reconfiguration
- `rezconfig.py` - Rez config
- `sync_package.py` - Package synchronization
- `run.bat` - Run script
- `wuwo.bat` - Wuwo batch script

## Usage

Run `wuwo.bat` to start the tool.
